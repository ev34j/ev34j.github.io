ev34j

A Java library for ev3dev devices.

Introduction

The Lego Mindstorms visual editor is a great first programming environment. However, as things get more complex, visual programming can get cumbersome and you want to use code rather than blocks. The ev34j-mindstorms library is meant to bridge the transition to code by providing simple Java objects that map directly to the Mindstorms blocks in the visual editor.

Example Code

import com.ev34j.mindstorms.motor.SteeringMotors;

public class SimpleSteeringMotors {

  public static void main(String[] args) {

    System.out.println("Trying out SteeringMotors");
    SteeringMotors motors = new SteeringMotors("A", "B");

    // Go forward 2 rotations at 25% power
    motors.onForRotations(0, 25, 2);
    motors.waitUntilStopped();

    // Go back 2 rotations at 25% power
    motors.onForRotations(0, -25, 2);
    motors.waitUntilStopped();
  }
}

Compatibility

The ev34j-mindstorms library works with these devices:

  • Lego Mindstorms - EV3
  • BrickPi
  • PiStorms
